\doxysubsection{Подробное описание}
Заголовок виджета Open\+GL для симуляции гравитационного взаимодействия. 

Данный файл объявляет:\+
\begin{DoxyItemize}
\item Структуру {\ttfamily \doxylink{struct_object}{Object}}, описывающую физическое тело в симуляции. Содержит данные о позиции, скорости, массе, радиусе, цвете и имени, а также методы для обновления состояния и отрисовки.
\item Класс {\ttfamily \doxylink{class_simulation_g_l_widget}{Simulation\+GLWidget}}, наследующийся от {\ttfamily QOpen\+GLWidget} и {\ttfamily QOpen\+GLFunctions}. Управляет рендерингом сцены, физикой гравитации (N-\/body simulation), камерой от первого лица и обработкой ввода пользователя.
\end{DoxyItemize}
