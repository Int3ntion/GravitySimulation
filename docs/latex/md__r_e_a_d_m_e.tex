\chapter{Gravity\+Simulation }
\hypertarget{md__r_e_a_d_m_e}{}\label{md__r_e_a_d_m_e}\index{GravitySimulation@{GravitySimulation}}

Симулятор гравитационного взаимодействия небесных тел с визуализацией через Open\+GL и интерфейс на Qt Widgets.

Данный проект демонстрирует:\+
\begin{DoxyItemize}
\item Работу с {\ttfamily QOpen\+GLWidget} и Immediate Mode ({\ttfamily gl\+Begin/\+gl\+End}).
\item Многопоточную логику физики (расчёт сил, обновление позиций планет, реализация гравитационной сетки).
\item Динамическое управение камерой (WASD, стрелки и пробел).
\item Взаимодействие с пользователем через диалоги добавления планеты или просмотра информации о планете.
\end{DoxyItemize}

Чтобы запустить проект, Вам необходимо иметь установленный {\bfseries{Qt6}} и {\bfseries{CMake}}.

\hypertarget{md__r_e_a_d_m_e_autotoc_md13}{}\doxysection{\texorpdfstring{Основные сценарии использования}{Основные сценарии использования}}\label{md__r_e_a_d_m_e_autotoc_md13}
\hypertarget{md__r_e_a_d_m_e_autotoc_md14}{}\doxysubsection{\texorpdfstring{Сценарий 1:\+ Запуск и осмотр сцены}{Сценарий 1:\+ Запуск и осмотр сцены}}\label{md__r_e_a_d_m_e_autotoc_md14}

\begin{DoxyEnumerate}
\item Запустите приложение ({\ttfamily Gravity\+Simulation.\+exe} или бинарный файл).
\item Вы увидите окно с несколькими сферами (планетами) и сеткой гравитационного поля.
\item Используйте клавиши {\bfseries{W, A, S, D}} для перемещения камеры по сцене, чтобы рассмотреть объекты с разных ракурсов (Если окно не реагирует на нажатие клавиш, нажмите на виджет с симуляцией левой кнопкой мыши).
\item Нажимайте {\bfseries{стрелки}}, чтобы вращать камеру вокруг центра сцены.
\end{DoxyEnumerate}

 {\itshape Рис. 1:\+ Главное окно, общий вид сцены с визуализацией гравитационной сетки и планетами.}\hypertarget{md__r_e_a_d_m_e_autotoc_md15}{}\doxysubsection{\texorpdfstring{Сценарий 2:\+ Добавление новой планеты}{Сценарий 2:\+ Добавление новой планеты}}\label{md__r_e_a_d_m_e_autotoc_md15}

\begin{DoxyEnumerate}
\item Найдите кнопку «Добавить планету» (или соответствующий пункт меню, если есть).
\item Откроется модальное окно {\ttfamily \doxylink{class_add_planet_dialog}{Add\+Planet\+Dialog}}.
\item Введите уникальное имя (например, "{}\+Mars"{}). Если имя занято, кнопка "{}\+OK"{} станет неактивной.
\item Задайте массу, радиус и начальные координаты.
\item Нажмите на кнопку {\bfseries{«Выбрать цвет»}}, чтобы открыть системный диалог и подобрать оттенок для новой планеты.
\item Подтвердите создание кнопкой «\+OK». Планета мгновенно появится на сцене и начнет двигаться согласно законам гравитации.
\end{DoxyEnumerate}

 {\itshape Рис. 2:\+ Интерфейс настройки параметров новой планеты.}\hypertarget{md__r_e_a_d_m_e_autotoc_md16}{}\doxysubsection{\texorpdfstring{Сценарий 3:\+ Редактирование и удаление объекта}{Сценарий 3:\+ Редактирование и удаление объекта}}\label{md__r_e_a_d_m_e_autotoc_md16}

\begin{DoxyEnumerate}
\item В интерфейсе найдите список планет. Дважды кликните по названию планеты, информацию которой Вы хотите узнать.
\item Откроется окно информации. Здесь вы можете изменить имя планеты.
\item Поля «Масса», «Координаты» и «Скорость» заблокированы для редактирования, так как они рассчитываются физическим движком.
\item Если объект больше не нужен, нажмите кнопку {\bfseries{«Удалить планету»}}. Объект исчезнет из симуляции.
\end{DoxyEnumerate}

 {\itshape Рис. 3:\+ Интерфейс просмотра информации о выбранной планете.}

\DoxyHorRuler{0}
\hypertarget{md__r_e_a_d_m_e_autotoc_md18}{}\doxysection{\texorpdfstring{Особенности реализации проекта:\+}{Особенности реализации проекта:\+}}\label{md__r_e_a_d_m_e_autotoc_md18}

\begin{DoxyEnumerate}
\item Интерфейс реализован без .ui файлов. Главное окно создаётся программно, чтобы избежать конфликтов версий Qt Designer и упростить понимание логики компоновки виджетов.
\item Для отрисовки сферы используется классический алгоритм генерации сферы через циклы и {\itshape gl\+Begin(\+GL\+\_\+\+QUAD\+\_\+\+STRIP)}.
\item Сетка гравитационного поля пересчитывается каждый кадр в {\itshape calculate\+Gravity\+Field()}. Для баланса производительности разрешение сетки ({\itshape m\+\_\+grid\+Resolution}) установлено на 30. Увеличение этого значения может {\bfseries{значительно снизить FPS}}.
\item Управление
\begin{DoxyItemize}
\item {\bfseries{W, A, S, D}} -\/ Перемещение камеры.
\item {\bfseries{Q, E}} -\/ Подъём/\+спуск камеры.
\item {\bfseries{Стрелки}} -\/ Вращение камеры.
\item {\bfseries{Пробел}} -\/ Пауза при зажатии.
\item {\bfseries{Двойной клик по планете в списке}} -\/ Редактирование имени, просмотр информации и удаление объекта.
\end{DoxyItemize}
\end{DoxyEnumerate}
