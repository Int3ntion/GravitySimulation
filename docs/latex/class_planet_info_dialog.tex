\doxysubsection{Подробное описание}
Модальное диалоговое окно для просмотра и частичного редактирования данных планеты. 

Предоставляет интерфейс для отображения детальной информации об объекте типа \doxylink{struct_object}{Object}:\+ название, масса, векторы позиции и скорости. Позволяет переименовать планету или запросить её удаление из симуляции.

Возвращает новое имя планеты, введенное пользователем в поле редактирования. 

Вызывается главным окном после закрытия диалога с результатом Accepted (OK), чтобы получить актуальное имя для обновления объекта в симуляции.

\begin{DoxyReturn}{Возвращает}
QString Строка с новым именем планеты. Если пользователь не менял имя, возвращается исходное значение.
\end{DoxyReturn}
Используется главным окном после успешного закрытия диалога (accept), чтобы обновить имя объекта в симуляции. Если пользователь ничего не изменил, возвращается исходное имя.
