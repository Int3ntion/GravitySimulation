\doxysubsection{Подробное описание}
Реализация диалогового окна для добавления новой планеты в симуляцию. 

Данный файл содержит реализацию класса \doxylink{class_add_planet_dialog}{Add\+Planet\+Dialog}, наследующегося от QDialog. Диалог позволяет пользователю задать все необходимые параметры нового небесного тела:\+
\begin{DoxyItemize}
\item Имя (с проверкой на уникальность и отсутствие пустых значений).
\item Массу, радиус и цвет объекта.
\item Начальные координаты и вектор скорости.
\end{DoxyItemize}

Окно является модальным, что предотвращает взаимодействие с основным интерфейсом до завершения ввода данных или отмены операции. Кнопка "{}\+OK"{} активируется только при корректном вводе имени. Выбор цвета осуществляется через стандартный диалог QColor\+Dialog.
