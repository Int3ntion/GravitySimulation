\doxysubsection{Подробное описание}
Исключение, сигнализирующее о нарушении логики работы симуляции во время выполнения. 

Выбрасывается внутри вычислительных методов (calculate\+Forces, calculate\+Gravity\+Field) при обнаружении аномальных состояний, которые делают дальнейшие расчеты невозможными или бессмысленными.

Сценарии использования:\+
\begin{DoxyItemize}
\item Попытка расчета гравитационного поля с недопустимым разрешением сетки (0 или отрицательное).
\item Обнаружение NaN или Inf в векторах скорости/\+позиции.
\item Деление на ноль в физических формулах. 
\end{DoxyItemize}
