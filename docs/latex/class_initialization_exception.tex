\doxysubsection{Подробное описание}
Исключение, выбрасываемое при неудачной инициализации компонентов симуляции. 

Возникает на этапе запуска приложения или создания виджета Open\+GL. Сценарии использования:\+
\begin{DoxyItemize}
\item Ошибка создания контекста Open\+GL.
\item Не удалось выделить память для буферов вершин.
\item Сбой инициализации таймера обновления кадра. 
\end{DoxyItemize}
