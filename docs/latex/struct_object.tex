\doxysubsection{Подробное описание}
Вспомогательный класс (структура данных), описывающий физическое тело в симуляции. 

Структура, представляющая физическое тело в гравитационной симуляции.

Хранит параметры планеты/\+объекта:\+ позицию, скорость, массу, радиус, цвет и имя. Содержит метод отрисовки сферы.

Хранит кинематические и динамические параметры объекта:\+
\begin{DoxyItemize}
\item Позиция и скорость (векторы из 3 координат).
\item Масса и радиус (скалярные величины).
\item Цвет (RGB) и имя (для отображения в UI).
\end{DoxyItemize}

Предоставляет методы для изменения состояния (ускорение, обновление позиции) и отрисовки объекта в виде сферы.
