\doxysubsection{Подробное описание}
Основной виджет приложения для симуляции и визуализации гравитации. 

Класс объединяет логику физического движка и рендеринг в реальном времени. Основные возможности:\+
\begin{DoxyItemize}
\item Реализация N-\/body симуляции (расчет гравитационных сил между всеми объектами).
\item Визуализация гравитационного поля в виде деформированной сетки.
\item Камера от первого лица с управлением через клавиатуру (WASD, стрелки, QE).
\item Интеграция с Qt через сигналы и слоты для уведомления UI об изменениях. 
\end{DoxyItemize}

Рассчитывает силы гравитационного притяжения между всеми объектами. 

Рассчитывает суммарную гравитационную силу, действующую на каждый объект.

Проходит по всем парам объектов, вычисляет силу по закону всемирного тяготения и применяет соответствующее ускорение к каждому объекту.

Реализует упрощенную модель N-\/body задачи. Для каждой пары объектов вычисляется сила притяжения по закону всемирного тяготения. Вводится ограничение минимального расстояния (150 единиц) для предотвращения численной нестабильности при столкновении. Ускорение применяется непосредственно к скорости объекта через метод accelerate. \Hypertarget{class_simulation_g_l_widget_aa5658c2d61593cb856b952c1523050f5}\index{SimulationGLWidget@{SimulationGLWidget}!calculateGravityField@{calculateGravityField}}

Пересчитывает значения гравитационного потенциала для визуализации сетки. 

Рассчитывает карту гравитационного потенциала для визуализации сетки.

Заполняет массив {\ttfamily m\+\_\+gravity\+Field}, суммируя вклад каждого объекта симуляции в каждой точке расчетной области. Используется упрощенная формула для визуализации "{}гравитационных ям"{}.

Создает двумерный массив m\+\_\+gravity\+Field размером (resolution+1)x(resolution+1). Для каждой точки сетки суммируется потенциал от всех объектов в симуляции. Формула потенциала упрощена для визуализации:\+ V = -\/A \texorpdfstring{$\ast$}{*} r /\+ sqrt(dx\texorpdfstring{$^\wedge$}{\string^}2 + dy\texorpdfstring{$^\wedge$}{\string^}2 + r\texorpdfstring{$^\wedge$}{\string^}2), где A зависит от логарифма массы. Результат смещается на -\/300 для лучшей видимости. \Hypertarget{class_simulation_g_l_widget_a842bc555e047a48292ad2956fb6cf685}\index{SimulationGLWidget@{SimulationGLWidget}!initializeGL@{initializeGL}}

Запускает симуляцию гравитационного взаимодействия. 

Запускает симуляцию, устанавливая шаг времени и запуская таймер.

Устанавливает шаг времени и запускает внутренний таймер, который вызывает обновление физики и перерисовку кадра.
