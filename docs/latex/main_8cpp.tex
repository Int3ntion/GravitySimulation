Данный файл содержит функцию \doxylink{main_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()}, которая является стандартной точкой входа для всех приложений на базе фреймворка Qt. Он отвечает за инициализацию приложения, создание главного окна и запуск цикла обработки событий.
