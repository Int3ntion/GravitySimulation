\doxysubsection{Подробное описание}
Заголовок диалогового окна для отображения и редактирования информации о планете. 

Данный файл объявляет класс \doxylink{class_planet_info_dialog}{Planet\+Info\+Dialog}, наследующийся от QDialog. Диалог предназначен для предоставления пользователю информации об объекте симуляции и позволяет выполнить следующие действия:\+
\begin{DoxyItemize}
\item Просмотреть массу, координаты и скорость объекта (только для чтения).
\item Изменить имя объекта.
\item Инициировать удаление объекта из симуляции.
\end{DoxyItemize}

Диалоговое окно является модальным, что гарантирует фокус внимания пользователя на информации о конкретной планете до принятия решения.

\begin{DoxyNote}{Заметки}
Класс зависит от структуры \doxylink{struct_object}{Object}, объявленной в \doxylink{_simulation_g_l_widget_8h}{Simulation\+GLWidget.\+h}. Поля координат и скорости заблокированы для редактирования, так как их изменение должно производиться исключительно физическим движком симуляции.
\end{DoxyNote}
