\doxysubsection{Подробное описание}
Реализация виджета Open\+GL для симуляции гравитационного взаимодействия и отрисовки 3D-\/сцены. 

Данный файл содержит реализацию класса \doxylink{class_simulation_g_l_widget}{Simulation\+GLWidget}, который наследуется от QOpen\+GLWidget. Он отвечает за:\+
\begin{DoxyItemize}
\item Инициализацию контекста Open\+GL и настройку проекции.
\item Расчет гравитационных сил между объектами (N-\/body simulation).
\item Построение карты гравитационного потенциала (сетка).
\item Управление камерой от первого лица.
\item Отрисовку планет в виде сфер и гравитационной сетки.
\end{DoxyItemize}
