\doxysubsection{Подробное описание}
Реализация диалогового окна для отображения и редактирования информации о планете. 

Данный файл содержит реализацию класса \doxylink{class_planet_info_dialog}{Planet\+Info\+Dialog}, который наследуется от QDialog. Окно предназначено для просмотра детальной информации об объекте симуляции:\+
\begin{DoxyItemize}
\item Название (доступно для редактирования).
\item Масса (только для чтения).
\item Текущие координаты (X, Y, Z).
\item Текущая скорость (X, Y, Z).
\end{DoxyItemize}

Также предоставляет функционал удаления планеты из симуляции через специальную кнопку. Все поля, кроме имени, заблокированы для изменения пользователем, так как они управляются физическим движком симуляции.

\begin{DoxyNote}{Заметки}
Диалог является модальным (set\+Modal(true)), что блокирует взаимодействие с главным окном до закрытия диалога.
\end{DoxyNote}
