\doxysubsection{Подробное описание}
Заголовок диалогового окна для добавления новой планеты в гравитационную симуляцию. 

Данный файл объявляет класс \doxylink{class_add_planet_dialog}{Add\+Planet\+Dialog}, наследующийся от QDialog. Диалог предоставляет пользовательский интерфейс для ввода всех необходимых параметров нового объекта:\+ имени, массы, радиуса, цвета, начальных координат и скорости.

Ключевые особенности класса:\+
\begin{DoxyItemize}
\item Валидация имени:\+ проверка на пустоту и уникальность среди существующих планет.
\item Блокировка кнопки "{}\+OK"{} до тех пор, пока имя не станет валидным.
\item Удобный выбор цвета через стандартный диалог QColor\+Dialog.
\item Широкий диапазон значений для физических параметров (масса, скорость, координаты), адаптированный под масштаб симуляции.
\end{DoxyItemize}

После успешного закрытия диалога (accept) вызывающий код должен использовать геттеры для получения данных и создания объекта структуры \doxylink{struct_object}{Object}.
