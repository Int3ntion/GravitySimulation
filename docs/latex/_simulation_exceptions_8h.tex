\doxysubsection{Подробное описание}
Определение пользовательских исключений для гравитационной симуляции. 

Данный заголовок содержит классы исключений. Все исключения наследуются от std::\+runtime\+\_\+error для совместимости со стандартной обработкой ошибок C++, но принимают QString для удобной локализации сообщений об ошибках и интеграции с Qt.

Классы исключений позволяют четко разделять типы ошибок:\+
\begin{DoxyItemize}
\item \doxylink{class_invalid_planet_data_exception}{Invalid\+Planet\+Data\+Exception}:\+ ошибки валидации входных данных (масса, координаты).
\item \doxylink{class_initialization_exception}{Initialization\+Exception}:\+ критические ошибки инициализации ресурсов (Open\+GL, таймеры).
\item \doxylink{class_simulation_logic_exception}{Simulation\+Logic\+Exception}:\+ ошибки внутренней логики симуляции (деление на ноль, некорректные параметры сетки гравитационного поля).
\end{DoxyItemize}
