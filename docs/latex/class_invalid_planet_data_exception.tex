\doxysubsection{Подробное описание}
Исключение, выбрасываемое при попытке создания планеты с некорректными данными. 

Используется в модулях ввода данных (например, \doxylink{class_add_planet_dialog}{Add\+Planet\+Dialog}) и валидации перед добавлением объекта в симуляцию. Сценарии выброса:\+
\begin{DoxyItemize}
\item Отрицательная масса или радиус.
\item Некорректные координаты (NaN, Inf).
\end{DoxyItemize}

Наследуется от std::\+runtime\+\_\+error. Конструктор автоматически конвертирует QString в std::\+string для совместимости со стандартным интерфейсом исключений.
