\doxysubsection{Подробное описание}
Основной класс приложения, управляющий симуляцией гравитации и пользовательским интерфейсом. 

Класс отвечает за:\+
\begin{DoxyItemize}
\item Инициализацию и компоновку UI (виджет отрисовки, список планет, слайдер G).
\item Обработку пользовательских событий (добавление планет, запуск/\+остановка симуляции).
\item Синхронизацию данных между логикой симуляции (\doxylink{class_simulation_g_l_widget}{Simulation\+GLWidget}) и отображением (QList\+Widget). 
\end{DoxyItemize}
