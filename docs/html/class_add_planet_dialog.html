Подробное описание

Модальное диалоговое окно для настройки параметров новой планеты.

Позволяет пользователю задать конфигурацию нового объекта симуляции. Управляет логикой валидации ввода, выбором цвета и компоновкой элементов интерфейса.

Конструктор класса AddPlanetDialog.

Инициализирует диалог, создает все элементы управления (поля ввода, спинбоксы, кнопки) и настраивает их начальные состояния. Подключает сигналы и слоты для обработки событий ввода текста и выбора цвета.

Аргументы
existingNamesСписок имен уже существующих планет. Используется для проверки уникальности имени новой планеты. По умолчанию пустой список.
parentУказатель на родительский виджет (по умолчанию nullptr).

Инициализирует модальное диалоговое окно "Добавить планету". Создает и настраивает все элементы интерфейса: поля ввода, спинбоксы, кнопку выбора цвета и кнопки управления. Настраивает диапазоны значений, шаги изменения и начальные значения параметров. Подключает сигналы и слоты для валидации имени и выбора цвета.

Аргументы
existingNamesСписок имен уже существующих планет, используемый для проверки уникальности имени новой планеты.
parentУказатель на родительский виджет (по умолчанию nullptr).
