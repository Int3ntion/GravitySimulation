Подробное описание

Основной виджет приложения для симуляции и визуализации гравитации.

Класс объединяет логику физического движка и рендеринг в реальном времени. Основные возможности:

  • Реализация N-body симуляции (расчет гравитационных сил между всеми объектами).
  • Визуализация гравитационного поля в виде деформированной сетки.
  • Камера от первого лица с управлением через клавиатуру (WASD, стрелки, QE).
  • Интеграция с Qt через сигналы и слоты для уведомления UI об изменениях.

◆ calculateForces

void SimulationGLWidget::calculateForces ( )
slot

Рассчитывает силы гравитационного притяжения между всеми объектами.

Рассчитывает суммарную гравитационную силу, действующую на каждый объект.

Проходит по всем парам объектов, вычисляет силу по закону всемирного тяготения и применяет соответствующее ускорение к каждому объекту.

Реализует упрощенную модель N-body задачи. Для каждой пары объектов вычисляется сила притяжения по закону всемирного тяготения. Вводится ограничение минимального расстояния (150 единиц) для предотвращения численной нестабильности при столкновении. Ускорение применяется непосредственно к скорости объекта через метод accelerate.

◆ initializeGL()

void SimulationGLWidget::initializeGL ( )
overrideprotected

Инициализация OpenGL контекста при первом создании виджета.

Инициализация контекста OpenGL при первом создании виджета.

Вызывается автоматически фреймворком Qt. Здесь производится:

  • Инициализация функций OpenGL.
  • Включение теста глубины (GL_DEPTH_TEST).
  • Создание начального набора объектов (Солнце, Земля, Метеор).
  • Расчет начального гравитационного поля.
  • Генерация сигнала planetListChanged.

Включает тест глубины (GL_DEPTH_TEST) для корректного перекрытия объектов. Заполняет начальный вектор объектов (Солнце, Земля, Метеор) и рассчитывает начальное гравитационное поле. Генерирует сигнал об изменении списка планет.

◆ paintGL()

void SimulationGLWidget::paintGL ( )
overrideprotected

Основной метод отрисовки кадра.

Основной метод отрисовки кадра (вызывается каждый раз при обновлении виджета).

Очищает буфер, применяет трансформации камеры, рисует гравитационную сетку и все объекты симуляции. Вызывается каждый раз, когда виджет нужно перерисовать.

Выполняет следующие действия:

  1. Очищает буфер цвета и глубины.
  2. Настраивает матрицу вида (ModelView) с учетом положения и вращения камеры.
  3. Отрисовывает сетку гравитационного поля (линии).
  4. Отрисовывает все объекты (планеты) как сферы.

◆ startSimulation

void SimulationGLWidget::startSimulation ( )
slot

Запускает симуляцию гравитационного взаимодействия.

Запускает симуляцию, устанавливая шаг времени и запуская таймер.

Устанавливает шаг времени и запускает внутренний таймер, который вызывает обновление физики и перерисовку кадра.

Шаг времени dt устанавливается равным 1e-7 (очень маленький шаг для стабильности). Таймер запускается с интервалом 16 мс, что соответствует примерно 60 FPS.

◆ stopSimulation

void SimulationGLWidget::stopSimulation ( )
slot

Останавливает симуляцию гравитационного взаимодействия.

Останавливает симуляцию.

Останавливает таймер обновления кадров. Если симуляция была приостановлена клавишей пробела, также сбрасывает шаг времени в ноль.

Если пробел не нажат, просто останавливает таймер. Если пробел нажат (режим паузы), устанавливает шаг времени в 0, чтобы физика не продолжала считаться даже если таймер вдруг сработает.

◆ updateObjects

void SimulationGLWidget::updateObjects ( )
slot

Выполняет полный цикл обновления состояния симуляции.

Обновляет состояние симуляции (физика + камера).

Последовательно вызывает:

  1. Расчет сил (calculateForces).
  2. Обновление позиций объектов.
  3. Пересчет карты гравитационного поля.
  4. Обновление камеры на основе ввода пользователя.
  5. Запрос перерисовки виджета.

Вызывается по таймеру. Последовательность действий:

  1. Расчет сил и обновление скоростей объектов.
  2. Обновление позиций объектов.
  3. Пересчет карты гравитационного поля.
  4. Обновление позиции камеры на основе нажатых клавиш.
  5. Запрос перерисовки виджета.
