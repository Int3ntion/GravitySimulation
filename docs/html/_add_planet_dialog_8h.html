Подробное описание

Заголовок диалогового окна для добавления новой планеты в гравитационную симуляцию.

Данный файл объявляет класс AddPlanetDialog, наследующийся от QDialog. Диалог предоставляет пользовательский интерфейс для ввода всех необходимых параметров нового объекта: имени, массы, радиуса, цвета, начальных координат и скорости.

Ключевые особенности класса:

  • Валидация имени: проверка на пустоту и уникальность среди существующих планет.
  • Блокировка кнопки "OK" до тех пор, пока имя не станет валидным.
  • Удобный выбор цвета через стандартный диалог QColorDialog.
  • Широкий диапазон значений для физических параметров (масса, скорость, координаты), адаптированный под масштаб симуляции.

После успешного закрытия диалога (accept) вызывающий код должен использовать геттеры для получения данных и создания объекта структуры Object.
